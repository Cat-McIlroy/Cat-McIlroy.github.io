About the Medicines Shortages Assistant

This application was developed by a PSI registered pharmacist with 11 years experience in the Irish community pharmacy sector. It is a free, publicly available tool which allows users to identify therapeutic alternatives to medications currently in short supply, or locate available pharmacy stock of same.

The aim of the application is to alleviate the ongoing problems caused by medicines shortages in Ireland. The hope is that this application will aid both patients and pharmacists, by making it easier for patients to source their prescribed medications, and reducing pharmacist workload by decreasing the volume of calls to pharmacies regarding medication shortages queries.
